-> Para executar o programa é preciso possuir o Python 3 instalado (Python 3.10 recomendado)

Após isso, no terminal, navegue até a pasta destino onde contem os arquivos:
    - compiler.py
    - LexicalAnalizer.py
    - terminals.py
    - program.txt

E execute o comando:
    python compiler.py
